# 二手房交易管理系统 java301

## 项目概述

本项目是基于Springboot和Vue的二手房交易管理系统，采用前后端分离的开发模式，整合了Java、SpringBoot、MyBatis-Plus、Redis等先进技术，旨在为用户提供一个高效、稳定的房产交易管理平台。

## 技术栈

- **开发环境：**
  - IntelliJ IDEA
  - JDK 1.8
  - Maven
  - Node.js

- **技术框架：**
  - Java
  - SpringBoot
  - MyBatis-Plus
  - Redis
  - Vue

- **数据库：**
  - MySQL（包含16张数据表）

## 系统角色

系统分为以下三种角色：

- 管理员
- 房东
- 用户

## 功能模块

### 登录模块

- 用户登录
- 用户注册
- 忘记密码

### 用户前台

- 首页轮播图
- 系统公告
- 房源信息
- 房源收藏
- 房源预约

### 后台管理

- 房东管理
- 用户管理
- 房源信息管理（可预约和查看评论）
- 看房预约管理
- 房型管理
- 支付管理
- 合同管理
- 城市管理
- 日常收费管理
- 系统公告
- 轮播图
- 公告分类管理

## 运行环境

- 服务器：推荐使用Linux服务器
- 数据库：MySQL 5.7及以上版本

## 部署步骤

（具体部署步骤将根据实际环境和配置需求有所不同，以下仅为基本步骤）

1. 准备运行环境（JDK、MySQL、Node.js等）
2. 部署后端服务（Springboot项目打包部署）
3. 部署前端服务（Vue项目打包部署）
4. 配置数据库连接
5. 运行系统

## 核心亮点

- **前后端分离**：系统采用前后端分离的模式，有利于模块化开发和维护。
- **技术先进**：基于Springboot和Vue的框架，确保系统的性能和稳定性。
- **高效管理**：后台管理功能强大，涵盖房源、用户、支付等全方位管理。
